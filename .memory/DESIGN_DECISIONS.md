# CDL 设计决策

## 语言设计

### 语法风格
- 类 CSS/JSON 的块结构 `{}`
- 关键字大写（Data, Chart）
- 中文友好（支持中文标识符）

### 注释
- 单行：`//` 或 `///`
- 多行：`/* */`

### 数据结构
```
Data <名称> {
    <列名1>,<列名2>,...
    <值1>,<值2>,...
    ...
}
```

### 图表定义
```
Chart <名称> {
    <数据源名称>
    -
    <图表类型>
}
```

## 架构决策

### 编译流程
```
Source Code
    ↓
Strip Comments (移除注释)
    ↓
Tokenize (分词)
    ↓
Parse (解析为 AST/CDLFile)
    ↓
Code Generation (代码生成 - 待实现)
```

### 当前支持的图表类型
- `line` - 折线图

### 计划支持的图表类型
- `bar` - 柱状图
- `pie` - 饼图
- `scatter` - 散点图
- `area` - 面积图
- ...更多

## 设计原则
1. **简洁** - 用最少的语法表达图表
2. **可读** - 非技术人员也能理解
3. **可扩展** - 易于添加新图表类型
4. **中文友好** - 原生支持中文
