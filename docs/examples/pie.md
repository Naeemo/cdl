# 饼图与环形图示例

```cdl
# 市场份额占比

| 厂商 | 占比 |
| --- | --- |
| A | 45% |
| B | 30% |
| C | 25% |

## pie

@title 市场份额分布

## 环形图

| 市场 | 30% |
| 运营 | 20% |
| 其他 | 10% |

## pie

@style ring
@color #4fc3f7
@title 预算分配
```
