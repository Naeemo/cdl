# CDL Renderer i18n - 国际化支持

CDL Renderer 现在支持完整的国际化（i18n）功能，包括图表标签、提示框、图例等多语言支持。

## 支持的语言

- 🇨🇳 简体中文 (zh-CN) - 默认
- 🇺🇸 English (en-US)
- 🇯🇵 日本語 (ja-JP)
- 🇰🇷 한국어 (ko-KR)
- 🇩🇪 Deutsch (de-DE)
- 🇫🇷 Français (fr-FR)
- 🇪🇸 Español (es-ES)
- 🇷🇺 Русский (ru-RU)
- 🇧🇷 Português (pt-BR)
- 🇮🇹 Italiano (it-IT)

## 快速开始

### 基本用法

```typescript
import { render, setRenderLocale } from '@naeemo/cdl-renderer-echarts';

// 设置语言
setRenderLocale('en-US');

// 渲染图表
const result = render(cdlFile);
```

### 按渲染设置语言

```typescript
import { render } from '@naeemo/cdl-renderer-echarts';

// 单次渲染使用指定语言
const result = render(cdlFile, {
  theme: 'dark',
  locale: 'ja-JP',
  i18n: true  // 启用国际化（默认）
});
```

### 自动检测浏览器语言

```typescript
import { render, detectBrowserLocale, setRenderLocale } from '@naeemo/cdl-renderer-echarts';

// 自动检测并使用浏览器语言
setRenderLocale(detectBrowserLocale());
```

## API 参考

### render(cdlFile, options)

渲染 CDL 文件为 ECharts 配置。

**参数：**
- `cdlFile` - CDL AST 对象
- `options` - 渲染选项（可选）
  - `theme`: 主题名称 ('light' | 'dark')
  - `locale`: 语言代码 ('zh-CN' | 'en-US' | ...)
  - `i18n`: 是否启用国际化（默认 true）

**向后兼容：**
```typescript
// 旧用法仍然支持
render(cdlFile, 'dark');  // 第二个参数可以是主题字符串
```

### setRenderLocale(locale)

设置全局渲染语言。

```typescript
import { setRenderLocale } from '@naeemo/cdl-renderer-echarts';

setRenderLocale('en-US');
```

### getRenderLocale()

获取当前渲染语言。

```typescript
import { getRenderLocale } from '@naeemo/cdl-renderer-echarts';

const currentLocale = getRenderLocale();  // 'en-US'
```

## 高级用法

### 自定义翻译

```typescript
import { configureI18n } from '@naeemo/cdl-renderer-echarts';

configureI18n({
  locale: 'zh-CN',
  translations: {
    'custom.key': '自定义文本',
    'label.value': '自定义数值标签'
  }
});
```

### 使用翻译函数

```typescript
import { t, formatNumber, formatDate } from '@naeemo/cdl-renderer-echarts';

// 简单翻译
const text = t('label.total', 'Total');

// 带插值的翻译
const drillText = t('tooltip.drillDown', 'Drill down to: {name}', { name: 'Category' });

// 格式化数字
const formatted = formatNumber(1234567.89);  // 根据语言自动格式化

// 格式化日期
const dateStr = formatDate(new Date(), { year: 'numeric', month: 'long' });
```

### 图表辅助函数

```typescript
import { 
  getChartTypeName,
  translateAxisName,
  translateSeriesName,
  createTooltipFormatter,
  createPieTooltipFormatter
} from '@naeemo/cdl-renderer-echarts';

// 获取图表类型本地化名称
const chartName = getChartTypeName('funnel');  // '漏斗图' / 'Funnel Chart'

// 翻译坐标轴名称
const axisName = translateAxisName('x');  // 'X 轴' / 'X Axis'

// 翻译系列名称
const seriesName = translateSeriesName('sales');  // 尝试翻译或原样返回

// 创建自定义提示框格式化器
const formatter = createTooltipFormatter({
  valueFormatter: (v) => `¥${v.toFixed(2)}`,
  nameFormatter: (n) => translateSeriesName(n)
});
```

## 在 CDL 中使用

CDL 支持通过 hints 设置图表语言：

```cdl
Chart 销售数据 {
    @locale "en-US"
    @title "Sales Overview"
    
    use SalesData
    type bar
    x month
    y amount
}
```

## 添加新语言

1. 在 `src/i18n/locales/` 创建新的语言文件（如 `th-TH.ts`）
2. 参考现有语言文件翻译所有键
3. 在 `src/i18n/locale-loader.ts` 中注册新语言

```typescript
// src/i18n/locales/th-TH.ts
export default {
  'chart.line': 'กราฟเส้น',
  'chart.bar': 'กราฟแท่ง',
  // ... 更多翻译
} as const;
```

## TypeScript 类型

```typescript
import { Locale, RenderOptions } from '@naeemo/cdl-renderer-echarts';

type Locale = 'zh-CN' | 'en-US' | 'ja-JP' | 'ko-KR' | 
              'de-DE' | 'fr-FR' | 'es-ES' | 'ru-RU' | 
              'pt-BR' | 'it-IT';

interface RenderOptions {
  theme?: string;
  locale?: Locale;
  i18n?: boolean;
}
```

## 注意事项

1. 语言设置是全局的，会影响所有后续渲染
2. 使用 `render(cdlFile, { locale })` 可以在单次渲染中覆盖全局语言
3. 如果某个翻译键不存在，将返回默认值或键名本身
4. 数字和日期格式根据语言自动调整（使用 Intl API）
