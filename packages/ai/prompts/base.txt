## CDL Syntax Reference

### Data Definition
```
@lang(data)
Data <Name> {
    <column1>,<column2>,...
    <value1>,<value2>,...
    ...
}
```

### SQL Data Source
```
@lang(sql)
@source('database_name')
Data <Name> {
    SELECT column1, column2 FROM table
}
```

### Chart Definition
```
Chart <Name> {
    use <DataName>
    type <chartType>
    x <xField>
    y <yField>
    [@style "description"]
    [@color "#hexcolor"]
}
```

### Supported Chart Types
- line: 折线图
- bar: 柱状图
- pie: 饼图
- scatter: 散点图
- area: 面积图
- radar: 雷达图
- combo: 混合图
- heatmap: 热力图

### Styling Hints
- @style: "平滑曲线", "带标记", "堆叠", "环形" 等
- @color: "#4fc3f7", "blue,red,green" 等
- @animation: "从左侧滑入" 等
